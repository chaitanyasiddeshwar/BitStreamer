# BitStreamer v1.5.0

Zero-transcode LAN media streamer for Fire TV. Audio is still bitstreamed untouched over HDMI — never transcoded, never remuxed.

## Highlights

- **Folder & multi-folder browsing** — point the server at one or more directories and browse them from the TV, instead of serving a single file.
- **Dolby Vision Profile 7 → 8.1 conversion** — keeps Dolby Vision on Fire TV (drops the enhancement layer, remaps the RPU) instead of falling back to HDR10; more robust DV7 handling, and DV now works in folder mode.
- **Resume playback** — pick up each title where you left off.
- **Seek-bar scrubbing previews** — trickplay thumbnails generated on demand with a live progress bar; smaller seek/preview step for short videos so left/right lands on the next frame.
- **Persistent thumbnail/preview cache** plus a `--no-caching` flag and faster ffprobe metadata.
- **Richer "stats for nerds"** — shows the active DV conversion (e.g. `dv7 → dv8`, `dv7 → hdr10`, `Native DV`), one reliable bitrate for video and audio, and live bandwidth.
- **`install-client.bat`** — one-step helper to sideload the client APK onto a Fire TV if you have adb.
- **Releases now include the User Manual PDF.**

## Notes

- Requires a Fire TV 4K / 4K Max for Dolby Vision; audio passthrough needs a real AVR/TV to verify the incoming format.
- Optional `ffmpeg`/`ffprobe` sidecars enable chapters, thumbnails, and previews; the server degrades gracefully without them.
